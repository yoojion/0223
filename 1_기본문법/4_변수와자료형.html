A. 변수의 종류
           - var
           - let
           - const
        B. 변수의 초기화
           1. 키워드를 작성
           var
           2. 식별자 선언
           var 변수
           3. 식별자에 값을 할당
           var 변수 = 0;
           1~3을 변수의 초기화라고 한다.
        c. let,const,var의 차이
           1. var
           -식별자 중복선언 가능
           -값의 재할당 가능
           -호이스팅 발생
           2. let
           -식별자 중복선언 불가능
           -값의 재할당 가능
           -호이스팅 발생 안 함
           3. const
           -식별자 중복선언 불가능
           -값의 재할당 불가능
           -호이스팅 발생 안 함
